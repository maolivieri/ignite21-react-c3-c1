## spacetraveling - Ignite
  Project created during the [@Rocketseat](https://www.rocketseat.com.br/) Ignite course as chapter III challenge.
  
  #### --- Features ---
  - Find a post.
  - Read the post.
  - Signup with Github.
  - Pay with Stripe to have access to full content.
  - Add comments to posts.
